What this key does

It drives your Sensor Dial actions from any key: a pedal, a G-key, a Multi Action step, or a plain key on another deck. Pick a command and, optionally, a target. The key shows a tick when the command reached at least one matching dial and an alert icon when none matched.

Give a dial a Link ID in its settings (under "Dial gestures & advanced") and put the same name here to steer just that dial or group of dials. The dial can be on a different Stream Deck than this key.
Only used by "Reset session stats". "Every dial, everywhere" ignores the Target and clears session stats on all dials, on all decks, including hidden ones.
Support
Copies a redacted diagnostics report (devices, data source, action states; no sensor values, no names) to the clipboard for a bug report. Nothing is uploaded.
