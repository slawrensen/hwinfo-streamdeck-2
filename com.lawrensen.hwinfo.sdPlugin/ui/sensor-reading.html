First time? HWiNFO setup

1. Install and start HWiNFO (hwinfo.com) in Sensors-only mode.
2. In HWiNFO Settings, enable Shared Memory Support. On the free version you can instead right-click sensors and tick “Report value in Gadget” (no 12-hour limit).
3. Pick a sensor below. That's it.

Sensor

Format
Pressing the key cycles current → min → max → avg.
Appearance
Pick a preset for this key only, or “Deck default” to follow the deck-wide theme (set under Advanced).
Layout
Two rows on one key: the sensor above on top, this one below. By default both rows show the same stat and the key press cycles them together, with one badge centered on the divider. Pin this row to a fixed stat for pairs like minimum and maximum; when the stats differ, each non-current row badges inline after its unit. Thresholds and the theme accent follow the first reading; the sparkline is a single-layout feature.
Cells fill in reading order: the sensor above top left, then second, third and fourth. Every cell shows the same stat (the press cycles all four together), and thresholds follow the first sensor, recoloring the whole key on alert.
Bar and Ring show where the value sits in its range: 0 to 100 for percentages, otherwise the values seen this session. Warn and critical thresholds mark amber and red zones toward the alarmed end: the high side normally, the low side when Direction alerts below.
Alerts
Warn at and Critical at use display units, the same numbers the key shows, °F included when Unit is on.
Advanced
Deck defaults (every key and dial)
Fallback for keys set to “Deck default”. Keys with their own theme keep it.
Deck-wide text intensity: Dim lowers the theme text, Custom uses your color. Keys and dials with their own Text setting keep it. Warn and critical colors always win.
