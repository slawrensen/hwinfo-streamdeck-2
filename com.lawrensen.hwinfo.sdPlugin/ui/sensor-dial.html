First time? HWiNFO setup

1. Install and start HWiNFO (hwinfo.com) in Sensors-only mode.
2. In HWiNFO Settings, enable Shared Memory Support. On the free version you can instead right-click sensors and tick “Report value in Gadget” (no 12-hour limit).
3. Pick a sensor below. That's it.

Sensor

Rotation
Leave the set empty to rotate through every reading of the picked sensor. Tick readings in the sensor list above to limit rotation to just those. Click a chip's name to rename that reading on this dial.
Steps through the rotation set (or the picked sensor's readings) on a timer. Runs even while turns are ignored.
View
The overviews list what rotation moves through: the rotation set, the active group, or the picked sensor's readings. Three rows marks the reading on the dial with a thumb on the left rail and keeps the shared name and session min/max on one context line (only the name shortens); two rows trades a row for big values and a trend sparkline.

Appearance
Pick a preset for this dial only, or “Deck default” to follow the deck-wide theme (set under Dial gestures & advanced).
Bar range
Blank fields follow the session low and high. The bar draws on the single view only.
Alerts
Warn at and Critical at use display units, the same numbers the dial shows, °F included when Unit is on.
Dial gestures & advanced
Gestures
Legacy: rotate cycles readings (pressed or not), push resets session stats, touch cycles the stat, long touch returns to current. Elite: plain rotate cycles readings without leaving the rotation set (or the picked sensor), pressed rotation jumps to the next rotation group (or the next sensor while no groups are defined), short push pauses the auto cycle, long push resets stats.
Switching Elite to Custom copies the Elite map into the selects below, so you can change just the one gesture you want different.
Remote control
HWiNFO Control keys whose Target matches this name steer this dial.
Deck defaults (every key and dial)
Fallback for keys and dials set to “Deck default”. Keys and dials with their own theme keep it.
Deck-wide text intensity: Dim lowers the theme text, Custom uses your color. Keys and dials with their own Text setting keep it. Warn and critical colors always win.
