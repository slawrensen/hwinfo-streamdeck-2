First time? HWiNFO setup

1. Install and start HWiNFO (hwinfo.com) in Sensors-only mode.
2. In HWiNFO Settings, enable Shared Memory Support. On the free version you can instead right-click sensors and tick “Report value in Gadget” (no 12-hour limit).
3. Pick a sensor below. That's it.

Dial gestures & advanced
Legacy: rotate cycles readings (pressed or not), push resets session stats, touch cycles the stat, long touch returns to current. Elite: plain rotate cycles readings without leaving the rotation set (or the picked sensor), pressed rotation moves to the next sensor, short push pauses the auto cycle, long push resets stats.
